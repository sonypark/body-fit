# 무브핏 (MoveFit) — 체형교정 운동 코치 🏃

앉아서 생활하는 학생을 위한 **게임형 운동 코치 웹앱**입니다.
키·몸무게·목표를 입력하면 체형(BMI)과 집중 부위에 맞는 운동을 유튜브 쇼츠와 함께 추천하고,
운동을 완료할 때마다 경험치가 쌓여 **파트너 캐릭터가 3단계까지 진화**합니다.

**▶ 바로 사용해보기**: https://sonypark.github.io/body-fit/

## 주요 기능

- 🧮 **BMI 기반 맞춤 추천** — 체형·목표·집중 부위에 맞는 운동만 노출, 과체중/비만은 고강도 운동 자동 제외(안전 필터)
- 📼 **유튜브 쇼츠 74개 연동** — 홈 추천 42개(Nobadaddiction 인기 영상) + 부위별 스트레칭 32개(자생한방병원 숏트레칭)
- 🎮 **캐릭터 성장** — 불(카이엔)·물(미스트랄)·풀(베인) 오리지널 스타터 중 선택, EXP를 모아 3단 진화
- 🔥 **출석·스트릭** — 연속 출석 기록, 월별 출석 달력
- 🪑 **쉬는 시간 10분 풀세트** — 목·허리·손목·다리 스트레칭 4종 자동 순차 진행
- ⏳ **운동별 쿨다운 3시간** — EXP 어뷰징 방지

## 구조

| 파일 | 내용 |
|------|------|
| `index.html` | 앱 전체 (HTML+CSS+JS 단일 파일, 빌드 도구 없음) |
| `체형교정_운동추천앱_기획설계서.md` | 기획/설계 문서 (기능 명세·추천 알고리즘·변경 이력) |

데이터는 서버 없이 **브라우저 localStorage**에만 저장됩니다.

## 로컬에서 실행

```bash
python3 -m http.server 8000
# http://localhost:8000/
```

> 유튜브 임베드는 http(s) 환경에서만 재생됩니다. 파일을 그냥 더블클릭(`file://`)하면 영상이 안 나옵니다.
